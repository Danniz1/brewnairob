# BrewNairobi 

A fully responsive, multi-page web application for **BrewNairobi** — a specialty coffee shop based in Nairobi, Kenya.

Built as a Web Application Development project using HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5.

---

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero, features, testimonials |
| About | `pages/about.html` | Our story, team, values |
| Menu | `pages/menu.html` | Coffee & food menu with filters |
| Gallery | `pages/gallery.html` | Filterable photo gallery |
| Blog | `pages/blog.html` | Articles and coffee guides |
| FAQ | `pages/faq.html` | Frequently asked questions |
| Reviews | `pages/testimonials.html` | Customer testimonials |
| Contact | `pages/contact.html` | Contact form + map |

---

## Features

- Responsive Bootstrap 5 layout (mobile, tablet, desktop)
- Dark / Light mode toggle (persisted in localStorage)
- JavaScript form validation with inline error messages
- Filterable gallery with category buttons
- Live blog search
- Scroll-to-top button
- Bootstrap Offcanvas mobile menu
- Bootstrap Accordion (FAQ), Modals (Gallery), Tooltips (Menu)

---

## Technologies

- **HTML5** — Semantic markup, ARIA attributes
- **CSS3** — Custom properties, Flexbox, Grid, media queries
- **JavaScript ES6+** — Modules, DOM manipulation, localStorage
- **Bootstrap 5.3** — Grid, components, utilities
- **Bootstrap Icons 1.11**
- **Google Fonts** — Playfair Display + Inter
